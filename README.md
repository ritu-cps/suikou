## Arduino の追加ライブラリについて

下記リンクのライブラリを入れてくれ
[Arduino 用ライブラリ LINK](https://drive.google.com/drive/folders/1qeW-CQY6QfRc4KwfLRRFvBdl0Zmwngdo?usp=sharing)

ライブラリのインストールについては次を参照してください：[URL](http://www.arduino.cc/en/Guide/Libraries)

#### 要約

ここにぶち込め(mac なら)

```cmd

```
